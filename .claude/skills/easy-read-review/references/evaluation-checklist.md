## 종합 평가 점수 산출 방법

1. ★ 필수 항목 통과 수 / ★ 전체 항목 수 × 70점 = 기본 점수
2. 일반 항목 통과 수 / 일반 전체 항목 수 × 30점 = 가산 점수
3. 기본 점수 + 가산 점수 = 종합 점수

> ★ 필수 항목이 하나라도 실패하면 최대 점수를 70점으로 제한한다.
