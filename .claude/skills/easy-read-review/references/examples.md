## 7. 쉬운 글 전환 실전 예시 (단계별)

### 7-1. 개인예산제 앱 문구

**원문 (일반 텍스트)**
> "개인예산제 서비스를 이용하시기 위해서는 지원 계획에 따른 예산을 등록하고, 실제 지출 내역을 입력하여 집행 현황을 관리하여야 합니다."

**Step 1: 보통 읽기(Plain Language) 수준**
> "개인예산제를 사용하려면, 먼저 예산을 등록해야 합니다. 그 다음 돈을 쓸 때마다 내용을 적어서 관리합니다."

**Step 2: 쉬운 글(Easy Korean) 최종**
> "내 예산 앱 사용 방법"
> 1. 예산을 넣습니다.
> 2. 돈을 쓸 때마다 적습니다.
> 3. 남은 돈을 확인합니다.

### 7-2. 공공후견제도 문서 (2017 안내서 사례)

**원문 (법률 문서)**
> "후견인은 피후견인의 재산관리 및 신상보호를 위하여 가정법원이 선임한 자로서, 피후견인의 법률행위를 대리하거나 동의·취소할 수 있다."

**쉬운 글 수정 결과:**
> **후견인**은 도움을 주는 사람입니다.
> 법원(판사)이 정한 사람입니다.
> 내가 중요한 것을 결정하고 선택할 때 나를 도와줍니다.

### 7-3. 서식 수정 예시 — 질문형 레이아웃 (2021 안내서 사례)

**원문 (공식 서식)**
> "성명: ______ 주민등록번호: ______ 장애유형: ______ 중증여부: □ 중증 □ 경증"

**쉬운 글(쉬운 용어 + 질문형 레이아웃):**
> **1. 나의 정보를 물어봅니다. 질문에 답을 적거나 골라 주세요.**
>
> 1) 나의 이름을 적어주세요.
> ________________________________
>
> 2) 나의 장애정도를 골라주세요.
> □ 중증 □ 경증

## 9. 취업 관련 쉬운 용어 (2021 안내서 부록 기반)

> 발달장애인이 취업 관련 서류에서 자주 접하는 어려운 용어를 쉽게 풀이

| 용어 | 쉬운 풀이 |
|:---|:---|
| 근로계약서 | 사장과 일하는 사람이 지켜야 하는 약속을 적은 서류 |
| 사업주 | 회사를 가지고 있는 사람 |
| 근로자 | 일하는 사람 |
| 임금 | 일해서 받는 돈 |
| 상여금 | 일을 잘했을 때 월급 말고 더 주는 돈 |
| 정규직 | 사장 마음대로 직장에서 쫓아내지 못하는 사람 |
| 구직신청서 | 일하고 싶은 곳에 일을 하고 싶다고 적는 서류 |
| 경력 | 내가 지금까지 했던 일들 |
| 동의서 | 어떤 일을 내가 허락하는 서류 |
| 서명 | 서류에 내 이름을 쓰는 것 |
| 개인정보 | 나에 대한 정보 (이름, 전화번호 등) |
| 필수 | 꼭 있어야 하는 것 |
| 해당없음 | 그런 적이 없다 |
