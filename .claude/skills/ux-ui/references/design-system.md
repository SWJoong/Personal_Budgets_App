# 디자인 시스템

## 색상 팔레트

| 역할 | 색상 | 코드 | 사용 |
|------|------|------|------|
| Primary | 파란색 | #2563EB | 주요 버튼, 강조 |
| Success | 초록색 | #16A34A | 완료, 달성 |
| Warning | 주황색 | #D97706 | 주의, 50% 초과 |
| Danger | 빨간색 | #DC2626 | 초과, 삭제 |
| Neutral | 회색 | #6B7280 | 보조 텍스트 |
| Background | 흰색 | #FFFFFF | 배경 |

> 모든 색상 조합은 대비율 4.5:1 이상 유지

## 타이포그래피

| 요소 | 크기 | 굵기 | 줄간격 |
|------|------|------|--------|
| 페이지 제목 | 24sp | Bold | 1.3 |
| 섹션 제목 | 20sp | SemiBold | 1.3 |
| 본문 | 16sp | Regular | 1.6 |
| 보조 텍스트 | 14sp | Regular | 1.5 |
| 버튼 | 16sp | Medium | 1.2 |

- 폰트: 나눔고딕 (한글), Inter (영문/숫자)
- 이탤릭·밑줄 사용 금지

## 간격 시스템 (4pt 기반)

4, 8, 12, 16, 24, 32, 48, 64px

## 컴포넌트 크기 기준

| 컴포넌트 | 최소 크기 | 권장 크기 |
|----------|-----------|-----------|
| 터치 버튼 | 44×44px | 48×48px 이상 |
| 텍스트 입력 | height 48px | height 56px |
| 탭바 아이템 | width 64px | |

## 아이콘 원칙

- 항상 텍스트 레이블과 함께 사용
- 크기: 24×24px (인라인), 32×32px (버튼 내)
- 의미: 직관적이고 보편적인 아이콘 선택
  - 예산: 💰 또는 지갑 아이콘
  - 지출: 📝 또는 영수증 아이콘
  - 홈: 🏠
  - 설정: ⚙️
