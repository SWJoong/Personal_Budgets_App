# 프로젝트 컨텍스트

## 프로젝트 개요
- **서비스명**: 개인예산제 앱 (Personal Budgets App)
- **목적**: 발달장애인이 자신의 예산을 스스로 관리할 수 있도록 돕는 교육·실습 웹 앱
- **핵심 가치**: 자기결정권, 쉬운 정보 접근성, 재정 자립

## 기술 스택
- **프론트엔드**: Next.js 15, React 19, TypeScript, Tailwind CSS
- **백엔드/DB**: Supabase (PostgreSQL, RLS, Auth)
- **배포**: Vercel
- **버전 관리**: GitHub
- **협업**: Slack, Google Workspace

## 주요 기능 영역
1. 예산 설정 및 관리 (월별/주별/일별/분기별)
2. 지출 입력 및 내역 조회
3. 예산 달성률 시각화
4. AI 기반 지출 인사이트
5. 사용자 인증 (Supabase Auth)

## 이해관계자
- **이용자**: 발달장애인 성인 (문해 수준 1–2)
- **보호자/지원자**: 부모, 활동지원사, 사회복지사
- **기관**: 복지관, 발달장애인지원센터, 장애인자립생활센터
- **개발팀**: PM, PL, FE, BE, QA, UX/UI, DevOps

## 접근성 기준
- KEAD 알기 쉬운 자료 제작 안내서(2021) 준수
- WCAG 2.1 AA 이상
- KS X 3253 모바일 앱 접근성 지침

## 현재 개발 단계
- UI/UX 개선 진행 중
- 쉬운 정보 접근성 검수 강화 예정
- AI 인사이트 기능 개발 예정
